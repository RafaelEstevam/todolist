## Inicialização do projeto

Para iniciar a aplicação é necessário instalar o Node.JS.
Após isso, na raiz do projeto front rodar o comando:

```
npm install

```
Para iniciar o front, na raiz do projeto front, no terminal rodar o comando:

```
nodemon

```

O ambiente deverá ser inicalizado em http://localhost:8080/

## Regras para criar o arquivo de importação

O "index" da tarefa "filha" deve ser maior que o "index" da tarefa "pai". É necessário também informar
o ID da tarefa pai.

Os IDs das tarefas não devem se repetir.
